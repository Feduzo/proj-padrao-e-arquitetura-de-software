# proj-padrao-e-arquitetura-de-software
Projeto de CRUD com IA
fiz esse projeto para um trabalho academico da USF - Universidade de são francisco
Seria uma integração basica de IA dentro de um CRUD, no caso a IA escolhi a ollama mistral

#Tutorial de como executar 
  1. Instale Ollama em https://ollama.com
  2. Em um terminal, execute: ollama run mistral
  3. Em outro terminal, instale: pip install requests
  4. Importe do meu repositório no Github o meu arquivo crud_ia_ollama_final.py
  5. Execute o meu arquivo em uma IDE.

OBS: Caso de erro cheque se o mistral esta rodando em um terminal aberto no mesmo ambiente, como no powershell ou no terminal de alguma IDE(vscode, Google Antigravity).
